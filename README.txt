CANTEEN FOOD CLUSTERING - PYTHON MINI PROJECT

DESCRIPTION:
-------------
This project clusters food items from a college canteen based on:
- Popularity
- Price
- Average Consumption Time

The clustering is done using KMeans and visualized using Matplotlib & Seaborn.

FEATURES:
---------
- GUI using Tkinter
- Run clustering with a button
- View the cluster scatter plot
- View clustered data in a table
- CSV and PNG outputs are saved automatically

REQUIREMENTS:
--------------
pip install pandas numpy matplotlib seaborn scikit-learn

HOW TO RUN:
-----------
1. Open a terminal or command prompt.
2. Run the following command:
   python canteen_clustering_gui.py

FILES:
------
- canteen_clustering_gui.py : Main Python GUI program
- canteen_clustered_menu.csv : Saved after clustering
- canteen_clusters_plot.png : Saved after plotting
